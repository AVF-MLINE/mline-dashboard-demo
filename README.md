# React дашборд для МИС (АГ + СД)

Демо-экран с фокусом на:

- нозологии АГ/СД;
- 3 режима отображения (`Сводный`, `Клинический`, `Операционный`);
- роли (`Врач`, `Управленец`);
- кнопку `Подготовить summary / сводку данных` с генерацией краткого AI-резюме по текущему контексту.

## Адреса версий

- `/` — страница сравнения версий;
- `/v1` — исходная версия дашборда;
- `/v2` — улучшенная версия с clinical/management блоками.
- `/v3` — оптимизированная версия с усиленной визуальной иерархией.

## Запуск

```bash
npm install
npm run dev
```
